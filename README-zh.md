# 🌟 星空Todo Pro

**一个美观实用的待办事项管理应用，带有番茄钟功能。**

![版本](https://img.shields.io/badge/version-v1.0.0-blue)
![许可证](https://img.shields.io/badge/license-MIT-green)

## ✨ 功能特点

- **📝 任务管理** - 轻松添加、完成、删除待办事项
- **🍅 番茄钟** - 25分钟专注计时器，帮助提升专注力
- **⏱️ 时间追踪** - 记录每个任务花费的番茄时间
- **💾 数据持久化** - 使用 localStorage 保存所有数据
- **🌌 视觉效果** - 星空主题设计，美观的动画效果
- **🌍 多语言支持** - 支持中英文，自动检测浏览器语言

## 🚀 快速开始

```bash
# 直接在浏览器中打开 index.html
open index.html
```

无需安装，支持离线使用！

## 📖 使用方法

1. 在任意现代浏览器中打开 `index.html`
2. 在输入框中输入任务，按 Enter 或点击"添加"按钮
3. 点击任务右侧的 🍅 按钮启动番茄钟
4. 点击 ✓ 按钮完成任务
5. 点击 🗑️ 按钮删除任务
6. 点击 🌐 按钮切换语言（自动检测浏览器语言）

## 🛠️ 技术栈

- HTML5 + CSS3 + 原生 JavaScript
- 无需安装任何依赖
- localStorage 存储数据
- CSS 动画实现视觉效果

## 🎨 界面预览

- 深色星空背景，带有动态星星效果
- 半透明毛玻璃效果的任务卡片
- 醒目的番茄钟计时器
- 任务统计展示（总番茄数、今日番茄数）
