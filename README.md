# 🌟 Star Todo Pro

**A beautiful and practical todo list app with Pomodoro timer.**

![Version](https://img.shields.io/badge/version-v1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Features

- **📝 Task Management** - Add, complete, and delete todos with ease
- **🍅 Pomodoro Timer** - 25-minute focus timer to boost productivity
- **⏱️ Time Tracking** - Track time spent on each task
- **💾 Data Persistence** - All data saved locally with localStorage
- **🌌 Beautiful Design** - Starfield theme with stunning animations
- **🌍 Multilingual** - Supports English and Chinese with auto-detection

## 🚀 Quick Start

```bash
# Just open index.html in your browser
open index.html
```

No installation required. Works offline!

## 📖 Usage

1. Open `index.html` in any modern browser
2. Type your task in the input box, press Enter or click "Add"
3. Click the 🍅 button to start the Pomodoro timer
4. Click the ✓ button to complete a task
5. Click the 🗑️ button to delete a task
6. Use the 🌐 button to switch language (auto-detects your browser language)

## 🛠️ Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript
- No dependencies required
- localStorage for data persistence
- CSS animations for visual effects

## 🎨 Preview

- Dark starfield background with animated stars
- Frosted glass effect task cards
- Prominent Pomodoro timer
